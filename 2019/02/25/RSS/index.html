AirtestIDE自动订阅RSS源

发表于 2019-02-25 | 分类于 爬虫 AirtestIDE

#0

以前经常在网站里会看到一个RSS的标志，一直不明白是什么东西，最近几天看到应用推荐网站看到了一个叫irreader的开源RSS和网站的订阅软件。在百度一搜，发现推荐这个软件的网站着实不少。我就借此机会来用一下RSS。

#1

irreader长这个样子

如图，这个软件自带了源市场，有很多可订阅源。我订阅了很多，但是我想在手机上也能看差不多的内容。（想要在手机端浏览irreader的RSS源）这时候就需要在手机上安装RSS阅读器，并且把irreader的源添加到手机app上。手机端让我满意的RSS阅读器叫Inoreader，这个软件第一次使用时要FQ才能注册，之后使用不需要再FQ。免费版最多可订阅150个源，专业版内置翻译功能，其余功能暂且不知。

#2

虽然irreader自带了导入导出功能（可导入导出为OPML类型的通用源列表），但手机端的app却并没有这个功能（PS：我试过其它手机端RSS阅读器，虽然有导入OPML源的功能，但是通过irreader导出的文件在手机端任何一个RSS阅读器都无法使用😂……）Inoreader自带源市场，但几乎都是英文内容，需要中文RSS源仅能手动输入URL地址添加自定义源，所以唯一的办法就是一个一个输入源URL并完成订阅。

irreader导出OPML文件格式如下，RSS源的地址是在xmlUrl字段：

OPML类型的文件可读性还是可以的，可以使用正则表达式来提取xmlUrl字段的内容，恰巧，最近在学习用AirtestIDE来爬区app内容的内容（其实AirtestIDE本来是由网易开发用来自动化测试的），然后就想到了它可以自动化操作手机，于是就有了思路：用Python读取OPML文件中每个RSS的URL，再利用AirtestIDE编写程序来依次完成：点击添加自定义源按钮->键入上一步获取到的URL->点击订阅按钮。开干。

#3

首先看看AirtestIDE的界面（此时已经通过USB线连接到了手机）

1、用来模拟人手对手机的操作
2、用以定位app界面控件和元素
3、Python代码区
4、软件运行的Log区，执行步骤和运行错误等等
5、手机屏幕内容显示区

  1. 手机打开USB调试模式（连续点击->关于手机-版本号->开启开发者选项，开启后在其中找到USB调试模式并打开）连接到电脑，允许授权。打开AirtestIDE软件，点击第五区域的刷新ADB（Android Debug Bridge）再点击connect连接手机，此时手机的屏幕就会显示在区域五：

  2. 读取OPML文件并提取其中RSS的URL链接：

    import re #导入正则表达式模块
pattern='.*?xmlUrl="(.*?)".*?' 
#编辑正则表达式.*?指在非贪婪模式（尽可能少匹配）下匹配任意非空白字符
with open('irreader-export-2-23.opml','r',encoding='utf-8') as f:
#使用with代码块，以只读模式打开文件，encoding='utf-8'便于打印输出
    res=re.findall(pattern,f.read())
    #re模块的findalll方法，匹配所有符合目标表达式的内容，返回一个列表
    print(res )#打印输出整个列表

  3. Q：如何做到精准点击添加自定义源按钮？ A：用AirtestIDE区域二（POCO辅助窗口）的功能（我是Android设备）

    1、app进入添加源界面，在区域二选择Android系统，在点击旁边的锁定图标（poco pause）。此时区域三（代码区）会弹出Poco mode has changed…… 点yes即可（代码会自动导入poco相关的库）

  4. 点击在区域五点击app右上角的添加自定义源，这时候是无法操作app的界面不会改变（因为加了锁😂）但是会显示出这个按钮的属性（敲黑板：对一个app来说，大多数控件的属性是唯一的）。并且，在区域二和区域四都发生了改变，区域二显示出了你当前点击的控件在这个app的控件树的哪个位置。区域四显示的是你当前选择的这个控件的几乎所有详细属性。我们常用区域四中出现的控件名字这个属性name：

  5. 在区域二中双击上一步选中的控件便可以自动生成poco代码（也可以手动写入，poco()的参数填区域四中的名名字即可），只需要再添加对应的poco的操作方法即可，这里就应该是点击方法（ 这里添加了.click() 方法）：

    poco("com.innologica.inoreader:id/action_add_subscription_url").click()

  6. 同理再分别定位到输入控件和订阅控件

    poco("android.widget.EditText").set_text(x)
#定位到输入文本框，.set_tetx(x) 往文本框中键入x，这里x代指RSS的URL地址
poco("android:id/button1").click()
#定位到“订阅”这个按钮控件，即可完成添加自定义RSS源

    #4

    完整代码：

    from airtest.core.api import * #IDE自动导入
import re #正则表达式库
from poco.drivers.android.uiautomation import AndroidUiautomationPoco 
#IDE自动导入
poco = AndroidUiautomationPoco(use_airtest_input=True, screenshot_each_action=False)
#IDE自动导入

pattern='.*?xmlUrl="(.*?)".*?'
with open('irreader-export-2-23.opml','r',encoding='utf-8') as f:
    res=re.findall(pattern,f.read())
    print(res)
    for x in res: # res是一个列表，for 循环抽出每个单独的RSS源地址
        poco("com.innologica.inoreader:id/action_add_subscription_url").click()
        poco("android.widget.EditText").set_text(x)
        poco("android:id/button1").click()
